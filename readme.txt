== Description ==

#1 editor for your content. Create and edit in WYSIWYG style responsive content based on Bootstrap framework.

You can find [Documentation](https://n1ed.com/docs/), [Demo](https://n1ed.com/demo/n1ed) and more detailed information about N1ED on [n1ed.com](https://n1ed.com/).

== Installation ==

1. Open the Plugins tab.
2. Add New -> Upload Plugin. Upload the archive with the module.
3. After successful installing, click Activate Plugin.
4. On the Settings tab click N1ED. Type in the API key from Dashboard. Click Save configuration.

You will find 'N1ED' menu in your WordPress settings.

For basic usage, you can also have a look at the [plugin web site](https://n1ed.com/demo/n1ed).
